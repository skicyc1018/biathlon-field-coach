# AUTO_BOARD_TEST_GUIDE — v1.2 Cost-Free Auto Board

무료 자동 보조(순수 JS) + 지도자 보정 구조의 검증 가이드입니다.
완전 자동 인식이 목표가 아니라, **자동 보조 후보를 지도자가 확인·보정**하는 흐름이 안정적으로 동작하는지 확인합니다.

---

## 0. 테스트 환경

- MacBook Safari / Chrome
- 로컬: `1_로컬_테스트_서버.command` 실행 → `http://127.0.0.1:8020/?v=12-costfree-stable`
- 개발자용 일괄 검증: `dev-test.html` (여러 사진을 한 번에 오버레이로 확인, 저장 없음)

## 1. 테스트 사진 목록 (직접 촬영/수집 권장)

| 번호 | 상황 | 기대 |
|---|---|---|
| A | 정면, 흰 배경, 자석 없음 | 외곽선/복사선/중심 후보가 거의 정확 |
| B | 정면, 파란 배경 | 배경색과 무관하게 검은 원 외곽선 검출 |
| C | 약간 기울어진 사진(회전) | rotation 반영, 타원이 보드에 맞음 |
| D | 중앙점이 자석으로 가림 | 중심이 검은 원 중심으로 잡힘(자석으로 끌리지 않음) |
| E | 손가락/장갑/클립이 일부 보임 | 손가락을 외곽선/자석으로 오인하지 않음 |
| F | 그림자 있음 | 그림자를 외곽선으로 오인하지 않음 |
| G | 자석 5개가 다양한 위치(명중/불명중 섞임) | 판정이 선 걸침 기준과 일치 |

> 사진은 앱이 저장/이동/삭제하지 않습니다. 메모리에서만 처리됩니다.

---

## 2. 자동 보조 합격 기준 (보조 수준)

- ⚡ 자동 보조 다시 실행 시 **빨간 점선/노란 외곽선/가짜 자석/가짜 AI 결과가 없어야** 한다.
- 가이드는 **입사 외곽선(얇은 청록), 복사선(밝은 청록), 중심 십자**만 표시한다.
- 라벨에 항상 **"자동 보조 후보(확인 필요)"** 가 보인다.
- 후보가 틀려도 다음이 **항상 가능**해야 한다.
  - ◎ 보드 맞춤 보정(이동/크기/회전/복사선 안·밖)
  - ① 중앙 / ② 복사선 / ④ 입사선 기준선 보정
  - ＋ 직접 마킹 / ↔ 점 이동 / × 점 삭제 / ○ 판정 수정

자동 보조의 정확도는 "참고용"이며, **틀려도 불합격이 아니다.** 보정으로 맞출 수 있으면 합격.

## 3. 판정 정답 기준

### 복사 (prone)
- 기준선 = 두 번째 굵은 실선(`rProneNorm`, 기본 0.50).
- 자석이 복사선 **안쪽 또는 선에 걸치면 명중**, 전체가 밖이면 불명중.
- 자석 중심점만 보지 않고 **자석 반경**을 고려한다.

### 입사 (standing)
- 기준선 = 검은 원 외곽선(`rStandingNorm` = 1.0).
- 자석이 검은 원 **안쪽 또는 외곽선에 걸치면 명중**, 검은 원 밖 배경이면 불명중.
- 배경색(흰/파랑/그림자)과 무관하다.

### 중앙 가림 사례 (D)
- 중앙점이 자석으로 가려지면 검은 원 중심을 중앙점으로 사용.
- 자동 보조 중심이 자석으로 끌리지 않는지 확인.

### 손가락/장갑 제외 사례 (E)
- 손가락/장갑/클립/로고/스티커/글자/그림자를 자석/기준선으로 인식하지 않아야 한다.

---

## 4. 직접 마킹 / 보정 합격 기준

- 사진 없이 **보드판 직접 마킹**에서도 탄착점 추가/이동/삭제/판정 수정이 된다.
- 자석 표시가 실제 자석을 큰 색으로 덮지 않는다(얇은 원 + 중심 점, 번호는 자석 바깥).
- 명중 = 청록, 불명중 = 빨강.
- 1~5발 모두 입력 가능.

---

## 5. 결과/오프라인 합격 기준

- 결과지에 홈으로 / 분석으로 / PDF 저장·인쇄 / 결과 이미지 공유 버튼이 **항상 보인다**(하단 메뉴에 가리지 않음).
- 리포트 항목: 선수, 일시, 구분, 발수, 세트, 명중 수, 명중률, 좌우/상하 평균 편차, 탄착군 크기, 중앙 평균 거리, 자동 보조 사용 여부, 지도자 수정 여부, 메모, 이미지.
- 홈 화면 추가 후 **오프라인에서 앱 실행 / 직접 마킹 / 결과 확인**이 된다.
- 네트워크 끊은 상태에서도 동작(외부 API 호출 없음).
