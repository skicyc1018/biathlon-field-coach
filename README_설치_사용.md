# Biathlon Field Coach v1.2 Cost-Free Auto Board

비용 없는(유료 클라우드 AI/외부 API 없음) 오프라인 현장형 사격 분석 PWA입니다.
무료 자동 보조가 보드판 가이드 후보를 제시하고, 지도자가 최종 확인·보정합니다.

## 핵심 동작
- ⚡ 자동 보조 다시 실행: 순수 JS로 보드판 가이드 후보만 표시(입사 외곽선/복사선/중심). 가짜 탄착점·가짜 AI 없음.
- ◎ 보드 맞춤 보정: 자동 후보가 틀리면 표준 보드 템플릿을 이동/크기/회전/복사선으로 맞춤.
- ① 중앙 / ② 복사선 / ④ 입사선 기준선 보정.
- ＋ 직접 마킹 / ↔ 점 이동 / × 점 삭제 / ○ 판정 수정 (사진 없이 보드판 직접 마킹도 가능).
- 복사 = 두 번째 굵은 실선 기준, 입사 = 검은 원 외곽선 기준, 자석 반경 고려해 선에 걸치면 명중.

## 설치 / 실행
- 로컬: `1_로컬_테스트_서버.command` 실행 → `http://127.0.0.1:8020/?v=12-costfree-stable`
- 또는 GitHub Pages 주소로 접속.
- 홈 화면에 추가하면 오프라인에서도 실행됩니다.

> ⚠️ **`index.html`을 파일로 직접 열지 마세요(`file://`).**
> `file://`로 열면 서비스워커(PWA 오프라인 캐시)가 등록되지 않고, 일부 브라우저에서 사진 처리/캐시가 제한됩니다.
> 반드시 **로컬 서버(`1_로컬_테스트_서버.command`)** 또는 **GitHub Pages(https://...)** 로 여세요.

## 좌표계 (v1.2 안정화)
- 모든 분석 좌표(탄착점·중앙·복사선·입사선·자동 보조 보드)는 **원본 이미지 좌표**(직접 마킹은 논리 좌표)로 저장됩니다.
- 화면에 그릴 때만 변환하므로 **화면 채움/창 크기 변경/가로·세로 회전**에도 탄착점이 사진에서 떨어지지 않습니다.
