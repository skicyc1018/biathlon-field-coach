# Biathlon Field Coach v1.0

야외 바이애슬론 사격장용 PWA 앱 1차 버전입니다.

## v1.0 기능

- iPhone/iPad 홈화면 설치용 PWA 구조
- 한 번 설치 후 앱 화면 오프라인 실행
- 선수 등록 및 기기 내부 저장
- 제로 / 복사 / 입사 구분
- 1~5발 세트 선택
- 사진 촬영 또는 업로드
- 손가락으로 탄착점 추가 / 이동 / 삭제
- 보드판 중앙점 수동 지정
- 자석을 덮지 않는 중심 십자 표시
- 선수별 결과 저장
- 결과지 화면 생성
- PDF 저장/인쇄용 출력 화면
- 결과 이미지 공유 또는 다운로드
- JSON 데이터 백업

## 중요한 점

v1.0은 `오프라인 수동 분석 기반`입니다.
Roboflow AI 자동분석은 다음 버전에서 중계 서버와 연결하는 구조로 추가합니다.

## MacBook에서 테스트하기

1. 압축을 풉니다.
2. `1_로컬_테스트_서버.command`를 더블클릭합니다.
3. Chrome 또는 Safari에서 아래 주소를 엽니다.

```text
http://127.0.0.1:8020
```

## iPhone/iPad에서 테스트하기

MacBook과 iPhone/iPad가 같은 Wi‑Fi 또는 핫스팟에 연결되어 있으면,
터미널에 표시되는 아래 형태의 주소로 접속합니다.

```text
http://<맥북IP>:8020
```

단, 이것은 테스트용입니다. 실제 배포용 오프라인 PWA는 HTTPS 주소에 올린 뒤 설치해야 합니다.

## 실제 지도자/선수 배포 방식

1. 앱 폴더를 HTTPS 웹주소에 업로드합니다. 예: 학교 서버, Netlify, Vercel, GitHub Pages 등.
2. 지도자와 선수가 iPhone/iPad Safari로 해당 주소를 엽니다.
3. 공유 버튼을 누릅니다.
4. `홈 화면에 추가`를 선택합니다.
5. 이후 홈 화면 아이콘으로 앱처럼 실행합니다.
6. 앱 화면, 선수 데이터, 사진, 수동 분석 결과는 기기 내부에 저장됩니다.
7. AI 분석과 카카오톡 공유는 데이터 또는 Wi‑Fi가 있을 때 사용합니다.

## 현장 사용 순서

1. 선수 등록
2. 분석 화면 이동
3. 선수 선택
4. 제로 / 복사 / 입사 선택
5. 발수 1~5발 선택
6. 사진 촬영
7. 점 추가 모드에서 자석 중심 터치
8. 점 이동 모드에서 손가락으로 보정
9. 필요 시 점 삭제
10. 중앙 지정 모드에서 보드판 중앙 터치
11. 결과 저장
12. 결과 화면에서 PDF 저장/인쇄 또는 이미지 공유

## 자석 표시 원칙

- 자석 전체를 큰 색상으로 덮지 않습니다.
- 자석 중앙에 작은 형광 청록색 십자만 표시합니다.
- 자석 바깥에 얇은 원을 표시합니다.
- 번호는 자석 바깥쪽에 표시합니다.
- AI 박스는 기본적으로 사용하지 않습니다.
