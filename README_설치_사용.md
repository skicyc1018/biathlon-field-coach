# Biathlon Field Coach v1.0.8

## 핵심 수정
- 홈 화면에 **보드판 직접 마킹** 버튼 추가
- 사진 없이 AI로 그린 표준 사격 보드판 위에 바로 탄착점 마킹 가능
- 분석 화면 확대 및 화면 채움 기능 유지
- 중앙점 자동 추정 개선: 가로 실선과 세로 실선 교차점을 우선 탐색하고, 자석이 가리면 표적판 외곽 원 중심으로 보정
- 기준선 자동 보정 기능 추가
  - ① 가는 점선
  - ② 복사 기준선: 두 번째 굵은 실선/기준선
  - ③ 가는 실선
  - ④ 입사 기준선: 검은 표적판과 흰 배경이 만나는 외곽선
- 직접 마킹, 점 이동, 점 삭제, 판정 수정 유지
- 자동 분석은 현재 오프라인 색상 기반 자석 후보 탐지이며, 지도자 보정이 필수

## 로컬 테스트
1. 압축 해제
2. `1_로컬_테스트_서버.command` 더블클릭
3. 브라우저에서 `http://127.0.0.1:8020/?v=108` 확인

`https://skicyc1018.github.io/biathlon-field-coach/?v=108`

## 현장 사용 기준
- 복사: ② 복사 기준선에 걸치면 명중
- 입사: ④ 입사 기준선 안쪽 또는 선에 걸치면 명중, 흰 배경으로 벗어나면 불명중
- 자석이 중앙 십자선을 가리면 `◎ 기준선 자동` 후 필요 시 `중앙 지정`으로 수동 보정
- 사진 없이 기록할 때는 홈 화면의 `보드판 직접 마킹` 사용
