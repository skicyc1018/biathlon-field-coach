AI 분석 수동 모드 v1.0.11은 직접마킹 보드판과 자동 복사선 보정

실제 보드판 기준선에 맞춰 명중·불명중을 판정하세요.

야외 사격장에서 Wi‑Fi가 없어도 선수·세트·사진·탄착점을 저장할 수 있습니다. 결과는 PDF 출력 화면으로 만들어 카카오톡 공유에 사용할 수 있습니다.

v1.0.11 범위

  • 사진 촬영과 앨범 업로드 버튼 분리
  • 분석 화면 확대/화면채움 모드 추가
  • 자동 분석 버튼 복구: 색상 기반 자석 후보 자동 탐지
  • 사격판 4개 원 기준선 재정의: 중앙 십자 → ① 첫 점선 → ② 두 번째 굵은 실선(복사) → ③ 세 번째 점선 → ④ 검은 원 외곽(입사)
  • v1.0.11: 직접 마킹 보드판은 실제 원 구조만 표시하고, ④ 입사선 추가 원을 제거
  • v1.0.11: 자동 분석 복사 기준선은 안쪽 점선이 아니라 ② 두 번째 굵은 실선 기준으로 보정
  • 복사: ② 두 번째 굵은 실선에 걸치면 명중, 입사: ④ 검은 원 외곽선 안쪽이면 명중
  • 직접 탄착 마킹, 이동, 삭제, 판정 수정 기능 유지
  • 사격 클릭 보정 제안 기능 추가
  • 업로드 사진을 한 화면에 맞춰 전체 보드판 확인 가능
  • 폰/패드 홈화면 설치용 PWA 구조
  • 선수 등록, 제로/복사/입사, 1~5발 세트 저장
  • 사진 촬영·앨범 선택, 손가락 탄착점 추가/이동/삭제
  • 자석을 덮지 않는 중심 십자 + 얇은 원 + 번호 표시
  • 결과지 출력 화면 및 PDF 저장용 인쇄 기능
