# 개발 메모 v1.0

## 앱 방향

MacBook 서버형 앱이 아니라 지도자와 선수가 각자의 iPhone/iPad에 설치해서 사용하는 PWA 앱으로 설계한다.

## v1.0 범위

- 오프라인 실행과 오프라인 저장을 우선한다.
- AI 자동분석은 다음 단계에서 연결한다.
- 현장에서는 AI가 실패해도 수동 기록이 가능해야 한다.
- 자석 표시는 판독을 방해하지 않아야 한다.

## 데이터 단위

선수 → 날짜/시간 → 사격 구분(제로/복사/입사) → 세트 → 1~5개 탄착점

## 분석 지표

- 입력 탄착점 수 / 계획 발수
- 탄착군 크기: 이미지 폭 대비 최대 탄착점 간 거리 비율
- 좌우 편차: 보드 중앙 대비 탄착군 중심의 좌우 위치
- 상하 편차: 보드 중앙 대비 탄착군 중심의 상하 위치
- 중앙 평균거리: 각 탄착점과 중앙점 거리의 평균

## 표시 색상

- 중심 십자: 형광 청록색 #00f5ff
- 보조 외곽선: 검정
- 중앙 보정선: 흰색/검정 점선
- 번호: 검정 배경 + 흰 글씨
